로스트아크 모든 떠돌이 상인 위치

2022년 8월 28일 일요일

로스트아크 모든 대륙 모든 지역 떠돌이 상인 (떠상) 위치 한페이지로 모아보기 로스트아크 통합 디스코드 떠돌이 상인 찾기. Ctrl + F 로 지역명 검색. 떠상 알림에 뜨는 지역명을 보고 빠르게 검색하기 쉽게 페이지 하나에 모았다. 아르테미스 떠돌이 상인 벤 로그힐 안게모스 산 기슭 국경지대 유디아 떠돌이 상인 루카스 살란드 구릉지 오즈혼 구릉지 루테란
