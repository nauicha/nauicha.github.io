윈도우10 hugo 블로그 만들기

2021년 2월 1일 월요일

깃허브저장소를 이용해 깃허브페이지 또는 Netlify 로 호스팅을 하려면 Git 윈도우버전을 설치해야한다. hugo 설치 Releases 페이지에서 다운로드. 다운받은 압축파일 폴더이름을 hugo로 바꾸어서 원하는 경로에 풀기. C:\hugo\bin 버전 업데이트 할 때는 hugo.exe 파일만 바꿔주면 된다. 환경변수 편집은 설정 > 시스템 > 정보 > 고급 시스템 설정 사용자 변수 Path 에 hugo 다운받은 경로를 새로 만들기로 추가한다. 설치 확인 hugo version 으로 설치 확인할 수 있다. 프로젝트 시작하기 hugo new site <이름> hugo 프로젝트
