윈도우10 hugo 블로그 만들기

2021년 2월 1일 월요일
  • hugo
  • windows

깃허브저장소를 이용해 깃허브페이지 또는 Netlify 로 호스팅을 하려면 Git 윈도우버전을 설치해야한다.

hugo 설치

Releases 페이지에서 다운로드.

다운받은 압축파일 폴더이름을 hugo로 바꾸어서 원하는 경로에 풀기. C:\hugo\bin
버전 업데이트 할 때는 hugo.exe 파일만 바꿔주면 된다.

환경변수 편집은 설정 > 시스템 > 정보 > 고급 시스템 설정

사용자 변수 Path 에 hugo 다운받은 경로를 새로 만들기로 추가한다.

설치 확인

hugo version 으로 설치 확인할 수 있다.

프로젝트 시작하기 

hugo new site <이름>

hugo 프로젝트 폴더 만들기

시작 

cd <이름>

프로젝트 폴더안에서 시작

테마 적용하기

hugo는 기본테마가 없으므로 시작할때 테마를 적용하고 시작해야한다. hugo 공식홈페이지 테마 목록

cd themes
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke

themes 폴더에 테마 저장소를 가져온다.

config.toml 

baseURL = "https://nasoru.com/"
languageCode = "ko"
defaultContentLanguage = "ko"
title = "nasoru"
theme = "ananke"

config.toml 에 theme = "테마(폴더)이름" 테마폴더에 저장된 테마의 폴더이름으로 저장

미리보기 

hugo server

localhost:1313 으로 실시간 로딩.

글쓰기

archetypes/default.md 파일이 기본으로 된 마크다운 파일이 만들어진다.
draft: true 는 지워야 미리보기나 빌드된 페이지가 보인다.

hugo new 경로/게시글주소/index.md

hugo 글쓰기 content 폴더만에 생성된다.

웹사이트 빌드 

hugo

hugo 사이트를 배포할 폴더 public 생성. 깃허브저장소로 public 폴더 연결하고 그 저장소를 깃허브페이지로 호스팅하면 된다.
