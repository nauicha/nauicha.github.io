USB 없이 윈도우 10 포맷하기

2021년 5월 2일 일요일
  • windows

시작하기 전

USB 없이 윈도우 10 포맷하는 방법이다.

8GB 이상인 비어있는 파티션이 필요하다. 네이버에 “윈도우 10 파티션 나누기” 검색. 현재 윈도우가 설치되지 않은 디스크

윈도우10 ISO 파일을 다운받아 압축을 푼다. Microsoft 공식 홈페이지에서 ISO파일을 다운받을 수 있다. 나는 반디집으로 압축 품.

윈도우10 USB 없이 포맷

윈도우10_하드설치.cmd 구글 드라이브 다운로드 링크 21/09/23 링크 재생성

위 스크린샷과 같이 boot 폴더와 sources 폴더만 비어있는 파티션 최상위 경로로 복사한다. 윈도우10_하드설치.cmd 파일을 다운받아 같은 위치에 옮겨두고 관리자 권한으로 실행한다.

관리자 권한으로 실행한 화면이다.

다시시작하면 Setup Windows 10 을 선택해서 윈도우를 설치 할 수 있다.

주의) 윈도우 포맷중 윈도우 설치 파일이 들어있는 드라이브를 포맷하면 안된다.

포맷 후

설치 중단 또는 cmd 중복실행해서 setup 선택화면이 뜨는 경우 Win + R 실행창에서 msconfig 부팅탭에서 삭제할 수 있다.
