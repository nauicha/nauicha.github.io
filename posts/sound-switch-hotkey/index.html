단축키(핫키)로 사운드 기본장치 변경하기 SoundSwitch

2022년 2월 8일 화요일
  • soundswitch

윈도우 10 에서 기본 재생장치를 바꾸려면 작업표시줄 아이콘을 클릭해야하고 디바이스를 선택하는데도 마우스가 필요하다. 그래서 키보드 단축키로 간단하게 전환할 수 있는 방법을 찾아보았다.

다운로드 https://soundswitch.aaflalo.me/
단축키로 간단하게 기본 재생(출력)장치를 전환할 수 있는 프로그램이다.
Hotkey enabled 체크, 단축키 지정.

상단탭 Playback 재생장치, Recording 녹음장치에서 전환할 장치 선택.

Audio Setting

  • Start automatically with Windows
    윈도우 시작시 자동실행. 작업스케줄러로 등록 하는게 더 빠른 것 같다.
  • Quick Menu on hotkey
    단축키가 입력되었을 때 마우스 위치에 퀵메뉴가 표시된다. 전환할 장치가 많을 때 편하다.

Notification

기본 장치가 변경될 때 알림표시

  • Windows Notification 화면 오른쪽 아래에 나타나는 윈도우 기본 토스트 알림. 여러번 연속으로 전환하면 딜레이 생김.
  • Banner 화면 왼쪽 위에 나타나는 알림.
  • 이 외 소리로 알림, 알림끄는 옵션있음.
