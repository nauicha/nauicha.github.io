WSL2 wget: unable to resolve host address

2022년 11월 26일 토요일
  • tag1
  • tag2

Windows에서 VPN에 연결하면 Bash에서 네트워크 연결이 끊어진다고 한다.

wget 설치 오류

wget: unable to resolve host address

sudo nano /etc/wsl.conf

resolv.conf 

[network]
generateResolvConf = false

wsl.conf 에 추가. Ctrl + x 로 저장.

wsl --shutdown
wsl

powershell 에서 wsl 종료 후 다시시작.

resolv.conf 

sudo nano /etc/resolv.conf

nameserver 8.8.8.8

resolv.conf 에 추가.

https://learn.microsoft.com/ko-kr/windows/wsl/troubleshooting#wsl-has-no-network-connectivity-once-connected-to-a-vpn
