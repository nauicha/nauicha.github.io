안드로이드에서 크롬 개발자 도구를 사용하는 방법

2021년 11월 17일 수요일
  • chrome
  • android

아래 방법은 안드로이드에서 크롬 개발자도구를 사용하는 것이 아니라 안드로이드를 PC에 연결해서 크롬 개발자도구를 원격으로 사용하는 방법. 안드로이드와 연결할 PC가 필요하다.

안드로이드 크롬에서 크롬 개발자 도구를 사용하기 위해 설정 – 휴대전화 정보 – 소프트웨어 정보 – 빌드 번호를 여러 번 눌러주면 안드로이드 개발자 옵션이 활성화 된다. 개발자 옵션에서 USB 디버깅 모드를 사용으로 바꿔준다. 컴퓨터에 USB로 연결

Minimal ADB and Fastboot 설치

출처 https://forum.xda-developers.com/t/tool-minimal-adb-and-fastboot-2-9-18.2317790/

다운로드 https://androidmtk.com/download-minimal-adb-and-fastboot-tool

adb devices

USB 디버깅 켠 상태로 연결하고 입력해주시면 위 스크린샷처럼 List of devices attached 목록에 표시됨.

chrome://inspect/#device

연결된 안드로이드에서 크롬을 실행하면 실행 중인 탭 목록이 표시됨.

안드로이드 크롬 화면이 PC 에서 표시됨. 연결완료.

PC 크롬에서 모바일 화면 보기

F12 크롬 개발자 도구에서 상단탭 요소 메뉴 왼쪽 버튼 또는 단축키 Ctrl + Shift + M 으로 화면폭을 숫자로 지정할 수 도 있고 소형 휴대기기 320px, 375px 대형 휴대기기 425px 태블릿 768px 노트북 1024px 대형노트북 1440px 4K 등 기기별로 페이지를 보여준다.
