VSCode 작업폴더 시작시 명령 실행하기

2022년 1월 29일 토요일
  • vscode

비주얼 스튜디오 코드 시작시 실행할 명령, 시작 작업 설정.

Ctrl + Shift + P 작업폴더에서 .vscode/tasks.json 파일 생성하기.

tasks.json 작업 설정하기 

{
  // See https://go.microsoft.com/fwlink/?LinkId=733558
  // for the documentation about the tasks.json format
  "version": "2.0.0",
  "tasks": [
    {
      "label": "ssh",
      "type": "shell",
      "command": "ssh nasoru@xxx.xxx.xxx.xxx",
      "windows": {
        "command": "ssh nasoru@xxx.xxx.xxx.xxx"
      },
      "presentation": {
        "reveal": "always",
        "panel": "new",
        "clear": true,
        "focus": true,
      },
      "runOptions": { 
        "runOn": "folderOpen",            
      }
    }
  ]
}
  • command 터미널 시작시 실행할 명령
  • windows 윈도우 운영체제 사용시 기본속성 대신 사용. 사용해야 정상작동해서 사용.

presentaion 사용자 인터페이스에서 작업 출력이 처리되는 방식

  • reveal always(기본) | never | silent 통합 터미널 패널을 앞으로 가져올지 여부
  • panel shared 터미널 공유 | dedicated 터미널 재사용 | new 새로운 터미널 사용
  • clear true | false(기본) 이 작업이 실행되기 전에 터미널을 지울지 여부
  • focus true | false(기본) 터미널이 입력 포커스 여부

runOptions 작업의 실행 동작을 지정

  • runOn default 명령으로만 실행 | folderOpen 자동으로 실행되도록 허용

https://code.visualstudio.com/docs/editor/tasks

자동시작 안될 때

  1. Ctrl + Shift + P 로 Tasks: Manage Automatic Tasks in Folder 한국어로는 작업: 폴더에서 자동 작업 관리 자동 작업 허용 필요.

  2. 작업 종료 팝업

Ctrl + Shift + P 로 Tasks:Run Task 아까 만든 작업 라벨이름 ssh 실행.
