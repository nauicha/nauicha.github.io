크롬 커서 깜빡임 캐럿 브라우징 해제하기

2022년 4월 15일 금요일
  • chrome

캐럿 브라우징은 마우스 대신 키보드를 이용해 페이지를 탐색할 수 있게 도와주는 기능이다. page up, page down, home, end, tab, 방향키 등… 하지만 이 기능을 사용하면 크롬 브라우저 사용 중에 텍스트 영역이 아닌 곳에서도 텍스트 커서가 깜빡인다. 비활성화 토글 단축키로는 F7 또는 설정페이지 고급 > 접근성 > 텍스트 커서로 페이지 탐색 을 해제한다.
