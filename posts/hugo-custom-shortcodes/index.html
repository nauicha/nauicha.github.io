Hugo 사용자 정의 단축코드 Shortcode 만들기

2022년 8월 30일 화요일
  • hugo
  • blog

Shortcode 는 Markdown 파일에서 사용할수 있는 단축코드로 긴 코드를 짧게 형식화해서 사용하거나 html 그대로 Markdown 으로 써진 글 안에 삽입할 수 있다.

Hugo 내장 단축코드 공식문서 기본으로 내장된 단축코드도 있다.

나만의 단축코드 Shortcode 만들기

최근에 자주 사용하고 있는 details 더보기를 단축코드로 만들어 보았다.

details.html

/layouts/shortcodes/ 경로에 details.html 이름으로 저장해야한다.

<details>
  <summary>{{ .Get 0 }}</summary>
  {{ .Inner }}
</details>

.Get 0 은 첫번째 인자를 가져옴. .Get 1 .Get 2 …

.Get "class" class 값을 가져옴.

.Inner 내용을 가져온다

[[<details 더보기이름>]]
내용
[[</details>]]

중괄호를 대괄호로 바꾸어야 한다. {{ <단축코드> }}

단축코드로 markdown 파일에 html 삽입하기

html.html 

{{ .Inner }}

html을 마크다운 파일에 삽입할 때 단축코드를 이용해 삽입한다.

[[<html>]]
내용
[[</html>]]

중괄호를 대괄호로 바꾸어야 한다. {{ <단축코드> }}
