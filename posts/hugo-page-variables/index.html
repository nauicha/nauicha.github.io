Hugo 페이지 변수 정리

2022년 8월 22일 월요일
  • hugo
  • blog

헷갈렸던 부분들만 정리했다.

페이지 종류 Kind

.Kind 

{{ .Kind }}

페이지의 종류를 출력 home, page, section, taxanomy 등..

.IsHome 

{{ if .IsHome }}
home
{{ end }}

사이트 루트 경로에서 True

.IsNode 

{{ if .IsNode }}
node
{{ end }}
  • 일반 콘텐츠 페이지에서 false.
  • index.md 로 된 페이지를 제외한 모든 페이지를 false
  • index.md 가 중첩되면 최하위 페이지를 True

.IsPage 

{{ if .IsPage }}
page
{{ end }}
  • 일반 콘텐츠 페이지에서 true

.IsSection 

{{ if .IsSection }}
section
{{ end }}

_index.md 파일로 된 페이지는 section

index.md 와 _index.md 의 차이

content 폴더에서 새로운 페이지를 생성할 때 index.md 와 _index.md 의 차이

index.md

일반 페이지 일반적인 블로그 글에서 사용. .IsSection 에서 False 반환. 동일한 폴더 이하 수준의 모든 리소스 사용 가능.

_index.md

트리 섹션 페이지. 페이지 중첩 가능. 하위 항목이 있는 페이지. .IsSection 에서 True 반환. 동일 폴더 수준내의 리소스만 사용가능.

  • _index
