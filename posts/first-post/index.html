네이버 블로그 html/css 수정안됨.
티스토리 블로그 자유로운 html/css 수정. 제거 불가능한 잡다한 스크립트.
워드프레스 대부분 자유롭게 커스터마이징 가능하지만 복잡한 이유로
hugo 정적사이트생성기 내가 필요한 기능만 깔끔하게 사용. + 가장 빠른 빌드 속도.

  • 마크다운 글쓰기
  • 간단한 구조
  • 필요한 기능만 사용가능

블로그 테마 만들기

간단한 1단 반응형.

  • hugo 이미지 리사이즈, 변환 위해서 content 폴더 구조 변경
content\posts
├─ post1
│   └─ index.md
├─ post2
│   ├─ image1.png
│   ├─ image2.png
│   ├─ image3.png
│   └─ index.md
├─ post3
│   ├─ image1.png
│   ├─ image2.png
│   ├─ image3.png
│   └─ index.md
  • 태그, 아카이브, 검색 페이지 추가.
  • 검색 기능을 추가했다. 제목, 태그, 요약에서 텍스트를 검색할 수 있다. 글 내용 전체를 검색할 수 있는 것은 아니다.
  • 모든 이미지 본문크기로 리사이즈, webp 로 변환. 이미지 클릭시 원본이미지 표시. 스크롤중에 새로고침시 이미지가 뜨지 않는 오류가 있다.
  • 목록 pagination 제거하고 더보기 버튼으로 변경.
