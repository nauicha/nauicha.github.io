윈도우11 애니메이션 제거, UI 패치, 최적화 개인설정 모음

2022년 7월 9일 토요일
  • windows

현재 설치된 버전 윈도우11 22000.778 아래 모든 설정은 개인적으로 생각했을 때 불편했던 부분들을 개선한 방법이다. 윈도우10을 포기하지 못한 것인지도 모른다. 파일 탐색기 탭 기능 기다리는중.

ExplorerPatcher

ExplorerPatcher 프로젝트 페이지. 작업 표시줄, 시스템 트레이, 파일 탐색기, 시작 메뉴등 윈도우 작업환경 개선을 목표로 하는 프로젝트. 내가 원하던 기능이 모여있다.

  • 둥근모서리를 각진모서리로 변경(창테두리만 적용). 개인적으로 둥근 모서리의 여백이 마음에 들지 않아 변경.
  • 시작 메뉴 맞춤 항목 제거. 공간 사용 가능. 맞춤항목을 숨기더라도 기본 윈도우11에서는 그 공간을 사용할 수 없고 3줄만 사용 가능하다.
  • 파일 탐색기 검색창 삭제. Everything 으로 대체. 사용하지 않던 검색창 삭제
  • 작업 표시줄 우클릭 메뉴 윈도우10 스타일로 복원. 작업표시줄 우클릭으로 작업관리자를 자주 사용했다.
  • 우클릭 메뉴 윈도우10 스타일 사용. 익숙하지 않다.
  • 초시계 표시 윈도우11에서 초시계 완전 제거됨.

윈도우11 애니메이션/시각효과 제거

  1. 설정 > 접근성 > 시각효과 > 애니메이션 효과 끄기.

  2. Win+R SystemPropertiesPerformance 실행.

내가 적용한 효과들. 체크된 항목은 끄면 많이 어색하다.

창의 제어 및 요소에 애니메이션 효과 주기 : 크롬 애니메이션 효과도 꺼버린다. scroll-behavior: smooth;

그 외 설정

  • 파일탐색기 설정 > 일반 > 내 PC 즐겨찾기 2항목 체크 해제.
  • 개인설정 > 시작 > 최근 항목 표시 해제. 시작메뉴 맞춤(추천) 비우기.
  • Win+V 자주 사용하는 클립보드 기능. 꺼짐(기본).
  • 네트워크 및 인터넷 > 프록시 > 자동설정검색 끄기. 인터넷정상 컴퓨터정상인데 크롬 실행(인터넷 연결)이 늦어진다면 이 기능이 켜져있다. 무조건 해제.

SysMain, Windows Search 서비스 끄기 

net stop SysMain
sc config SysMain start=disabled
net stop WSearch
sc config WSearch start=disabled

SSD를 사용한다면 효과가 없는 서비스. 백그라운드에서 CPU를 항상 사용.

개인정보데이터수집 차단 

reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\DataCollection" /v "AllowTelemetry" /t REG_DWORD /d "0" /f

전원옵션 고성능 

POWERCFG -SETACTIVE 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c
POWERCFG /Change /standby-timeout-ac 0
POWERCFG /Change /disk-timeout-ac 0

최대 절전모드 끄기 

powercfg -h off
