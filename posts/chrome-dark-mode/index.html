크롬 UI 다크모드 적용하기

윈도우10에서 크롬을 다크모드로 바꾸는 방법들이다. 아래 방법들은 크롬 UI 를 어둡게 해주는 것이고 웹 컨텐츠를 다크모드로 사용하려면 웹 페이지에서 다크모드를 지원하거나 또 다른 방법이 필요하다.

1. 윈도우 설정으로 다크모드

바탕화면 우클릭 개인설정에서 선택. 사용자 지정으로 선택되어있지만 기본 앱 모드 선택이 뜨지 않는 경우는 다른 옵션으로 선택 후 다시 사용자 지정으로 선택하면 된다.

유튜브처럼 기기테마를 감지할 수 있는 사이트는 다크테마로 적용됨. 이렇게 했을 때 단점은 크롬 뿐만 아니라 윈도우에서 쓰는 모든 기본 앱들이 다크모드로 표시된다.

2. 크롬 테마로 다크모드

크롬 웹 스토어 에서 테마를 선택하는 방법이 있다. dark 로 검색하면 더 많은 관련 테마를 볼 수 있다. 장점은 사용자가 다크테마 스타일을 선택할 수 있지만 크롬 테마를 설치해야하고 기기 테마를 다크모드로 감지하지 못한다.

3. 크롬 시작옵션으로 다크모드

윈도우 설정으로 기기 테마를 바꾼 것에서 크롬만 적용 가능한 방법이다.

-force-dark-mode

크롬 속성 > 바로가기 대상 입력란에 위 텍스트를 추가해주고 적용한다. 이렇게 실행하면 기기테마를 다크모드로 감지할 수 있다. 작업표시줄이 아닌 다른방법으로 크롬 실행이 적용안됨.

크롬 웹 컨텐츠 다크모드 적용하기

지금까지는 크롬 UI 부분만 다크모드로 바뀌었다. 웹 페이지에서 다크모드를 지원한다면 모르겠지만 지원하지 않는 사이트들은 강제로 적용할 수 있다.

1. 크롬 실험실에서 다크모드

chrome://flags/#enable-force-dark 크롬 실험실에서 제공하는 다크모드이다. Enabled로 적용후 Relaunch. 이 방법이 가장 자연스럽게 다크모드로 보여주는 것 같다. 하지만 토글기능이 없어서 매번 다크모드를 전환할 때마다 위 과정을 반복해야한다.

2. 크롬 확장프로그램으로 다크모드

크롬 웹 스토어 에서 검색하면 많은 다크모드 확장프로그램들이 있다.

대부분 확장프로그램에서 토글버튼으로 고정할 수 있다. 여기서 사용한 확장프로그램은 Dark Mode.
단점은 강제로 다크모드로 적용한 것이기 때문에 버튼이나 링크들이 보이지 않을 수 있다.
