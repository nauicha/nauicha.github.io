리눅스 Screen 터미널 명령어/단축키

2022년 2월 18일 금요일
  • linux
  • screen

백그라운드에서 터미널을 실행. 연결 끊기지 않고 작업을 실행할 때 가상 터미널 생성. 실행중인 프로세스는 백그라운드에서 계속 실행됨.

설치 

sudo apt update
sudo apt install screen
sudo yum install screen

Ubuntu 에서 Screen 설치
CentOS 에서 Screen 설치

명령어 

screen -version # screen 설치확인
Screen version 4.06.02 (GNU) 23-Oct-17

screen -S <이름> # 이름으로 터미널 세션 만들고 시작

screen -ls # 실행중인 세션 목록

screen -r <이름> # 분리된 세션으로 연결
screen -d <이름> # 세션 분리하기
screen -d -r <이름> # 연결된 세션 분리하고 다시연결

단축키

  • Ctrl + a 누르고 d 세션에서 분리
  • Ctrl + a 누르고 k (y/n) 확인 / Ctrl + d 세션 종료.
