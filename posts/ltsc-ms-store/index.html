Microsoft 스토어 설치

윈도우 10 Enterprise LTSC (Long-term-Servicing-Channel) 에서는 윈도우에서 기본적으로 제공하는 앱과 MS 스토어등이 없는데 설치하는 방법.

테스트 환경 Windows 10 Enterprise LTSC 21H2 build 19044.1466
다운로드 https://github.com/kkkgo/LTSC-Add-MicrosoftStore/releases

관리자 권한으로 실행. DesktopAppInstaller / StorePurchaseApp / XboxIdentityProvider 포함됩니다.
설치를 원하지 않으면 각각의 appxbundle 삭제

오류가 나면 컴퓨터 다시시작 후 재시도. (윈도우 설치 후 처음 실행했을 때 오류, 재부팅 후 정상설치)

설치, 실행 테스트 완료.

캡처, 클립보드 기능

LSTC 에서는 Win + Shift + S 캡처 기능, Win + v 클립보드 기능이 작동하지 않습니다. MS 스토어에서 캡처 및 스케치 앱을 설치.
