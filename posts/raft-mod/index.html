Raft 래프트 모드 적용하는 방법(치트)

2021년 8월 10일 화요일
  • game
  • raft
  • cheat
  • steam

Raft 스팀게임 숨겨진 업적 중에 비행기섬이랑 난파선섬 찾기를 할 때 사용. 모드로더를 실행하려면 스팀 Raft를 설치해야함.

ModLoader 다운로드

Raft 모드로더 사이트에서 다운로드 받는다.

다운로드 받은 RMLLauncher.exe 을 실행하고

업데이트를 시켜준다.

모드로더 메인화면. UI가 잘려서 보임.

모드로더 Start 버튼으로 래프트를 실행해준다. 스팀에서 실행하면 안됨.

모드 적용하기

  • 모드로더 홈페이지 모드목록
  • 모드로더 홈페이지 ItemSpawner

이 글에서는 Raft ItemSpawner 라는 모드를 설치한다.

크롬으로 모드 페이지에서 오른쪽 하단의 Install Mod 초록색 버튼으로

열어주고 Install it 버튼으로 설치해준다. (이 UI도 짤림..)

MOD MANAGER 탭에서 모드오른쪽 플러그 아이콘으로 적용가능. 자동로드기능은 느낌표 아이콘. STATUS 가 Running… 이라고 바뀌면 모드 로딩완료. 월드에 들어가서 모드 즐기기.

설치한 모드의 명령어를 사용하기 위한 콘솔창 F10

설치한 모드 관리 창 F9

Item Spawner 모드

모드 메뉴 여는 단축키는 F8

raft의 모든 아이템, 동물, 작은섬, 대형섬 소환가능하다. 동물 킬, 비행기섬, 난파선섬 등 업적 잠금해제하기에 매우 편리하다.
