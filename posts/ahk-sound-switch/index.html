단축키(핫키)로 기본 재생장치 변경 스크립트 AHK

2022년 2월 10일 목요일
  • soundswitch
  • autohotkey

이전 글 - SoundSwitch 에서는 장치 변경알림 배너나 윈도우토스트알림이 전체화면에서 보이지 않는다.

오토핫키 스크립트

  • +F10 은 Shift + F10 을 단축키로 설정. 다른 단축키로 설정하기 위한 관련 문서
  • nircmd 가 필요하다. 같은폴더에 저장.
;#NoTrayIcon
#Persistent
#SingleInstance

^!BACKSPACE::
	toggle:=!toggle
	if toggle	{
		Run nircmdc setdefaultsounddevice "스피커"
		soundToggleBox("스피커")
	}	
	else {
		Run nircmdc setdefaultsounddevice "Headset Earphone"
		soundToggleBox("Headset Earphone")
	}
Return

soundToggleBox(Device){

	IfWinExist, soundToggleWin
	{
		Gui, destroy
	}
	
	Gui, +ToolWindow -Caption +0x20000 +alwaysontop
	Gui, Margin, 0, -8
	Gui, color, C242424
	Gui, Font, S11 CFFFFFF, 맑은 고딕
	Gui Add, Text, x+m w300 h70 +0x201, 기본 장치 변경 - %Device%
	SysGet, screenx, 0
	SysGet, screeny, 1
	xpos:=(A_ScreenWidth/2) - 150
	ypos:=screeny-150
	Gui, Show, NoActivate x%xpos% y%ypos% w300 h70, soundToggleWin

  SetTimer,soundToggleClose, 2000
	WinSet, Region, 0-0 w350 h30 R30-30

}

soundToggleClose:
	SetTimer,soundToggleClose, off
	Gui, destroy
Return

exe 파일로 만들어서 포맷할 때마다 항상 작업스케줄러, 로그온 시 실행으로 등록. nircmdc.exe .
