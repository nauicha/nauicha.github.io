윈도우10 SFTP 네트워크 드라이브에 연결하기

2022년 1월 24일 월요일
  • windows
  • sftp

원격 디렉토리를 윈도우10 탐색기에 마운트하는 방법.

VSCode Remote-SSH 확장은 서버에 VS Code server (최소 1GM RAM, 2 CPU cores 필요) 를 설치하는 대신 네트워크 드라이브로 마운트하고 간단한 작업폴더로 사용하기 위해 알아보았다.

터미널은 CMD OpenSSH 연결해서 사용. (윈도우10 선택적 기능에 포함된 OpenSSH 클라이언트 필요)

ssh username@100.100.100.100 -i ~/.ssh/path

ssh 키파일로 연결. -i

ssh -L 1313:100.100.100.100:1313 username@100.100.100.100 -i ~/.ssh/path

ssh 포트 1313 연결 추가. -L

준비물

  • OpenSSH 클라이언트 윈도우10 선택적 기능
  • WinFsp, https://github.com/billziss-gh/winfsp/releases/
  • SSHFS-Win, https://github.com/billziss-gh/sshfs-win/releases/

WinFsp, SSHFS-Win latest 최신버전 설치.

사용방법

네트워크 우클릭 메뉴에서 네트워크 드라이브 연결(N)…

\\sshfs\username@HOST[!PORT][\PATH]

PORT 기본 22, PATH 드라이브에 연결될 루트경로.

비밀번호 입력으로 연결.

\\sshfs.k\username@HOST[!PORT][\PATH]

id_rsa 키파일 경로 %USERPROFILE%/.ssh/id_rsa id_rsa 파일은 puttygen 에서 ppk 파일으로부터 가져오기가능. ppk 또는 pem 키파일은 https://github.com/evsar3/sshfs-win-manager/releases/ GUI 프로그램 필요.

RaiDrive

지금까지는 WinFsp, SSHFS-Win, sshfs win manager 3개가 필요했지만 RaiDrive 하나로 더 많은 기능을 편하게 이용할 수 있다.

https://www.raidrive.com/

Raidrive 는 구글드라이브, 원드라이브, 드롭박스, 네이버 MYBOX, MEGA 등 클라우드를 내컴퓨터 디스크로 연결 시켜준다. + WebDAV, SFTP, FTP 포함.

무료버전 사용시 최대 8개 드라이브까지 추가할 수 있고 드라이브 목록,처음 프로그램 실행할 때 광고가 표시된다.

유료버전 사용시 광고를 제거하는데 월 2$ 읽기 가능한 저장소를 개당 월 $0.5로 판매하고 있기 때문에 추천하지는 않는다.
